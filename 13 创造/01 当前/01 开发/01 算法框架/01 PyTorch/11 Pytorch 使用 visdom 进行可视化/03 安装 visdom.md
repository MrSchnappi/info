# 安装 Visdom

在 python 3环境下，使用安装 Visdom 模块。

```
pip install visdom
```

启动服务器：

```
python -m visdom.server
```


一旦启动服务器，就可以通过在浏览器中输入 http://localhost:8097 来访问 Visdom。

localhost 可以换成 Visdom 服务的托管地址。<span style="color:red;">嗯，好的。可以换成 visdom 服务的托管地址。</span>
