# Git 多个 key 的使用

当需要连接 github 的多个 repo 的时候，肯定要对应每个 repo 有不同的 key，那么就需要每次在某个 repo 里提交的时候，要先把 key 加进去：

```
ssh-agent bash
ssh-add /c/Users/evo/.ssh/id_rsa_hugo
```
