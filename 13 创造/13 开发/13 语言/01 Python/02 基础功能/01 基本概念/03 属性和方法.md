## 属性和方法

属性（Attributes）指在当前这个 object 里，还有一些其他的 Python object。方法（method）指当前这个 object 自带的一些函数，这些函数可以访问 object 里的内部数据。

通过`obj.attribute_name`可以查看：

```
a = 'foo'
a.<Press Tab>
```

可以通过`getattr`函数来访问属性和方法：

```
getattr(a, 'split')
```

输出：

```
<function str.split>
```
