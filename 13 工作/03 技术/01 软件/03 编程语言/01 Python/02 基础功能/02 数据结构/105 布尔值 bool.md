# 布尔值 bool



布尔值和布尔代数的表示完全一致，一个布尔值只有`True`、`False`两种值，要么是`True`，要么是`False`。

在判断的时候，只要是非零数值、非空字符串、非空 list 等，就判断为`True`，否则为`False`。


## 布尔值的表示

在 Python 中，可以直接用`True`、`False`表示布尔值（请注意大小写）。

## 布尔值的运算
