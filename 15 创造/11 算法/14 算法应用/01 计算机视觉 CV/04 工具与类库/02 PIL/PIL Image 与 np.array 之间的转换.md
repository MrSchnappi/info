# PIL Image 与 np.array 之间的转换

Open I as an array:

```
I = numpy.asarray(PIL.Image.open('test.jpg'))
```


Do some stuff to I, then, convert it back to an image:

```
im = PIL.Image.fromarray(numpy.uint8(I))
```
